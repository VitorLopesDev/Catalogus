 Catalogus
API REST para gerenciamento de acervo de biblioteca, desenvolvida como projeto acadêmico em equipe utilizando Java e Spring Boot.
 Tecnologias

Java 21
Spring Boot
Spring Data JPA
H2
Maven
Git/GitHub

 Funcionalidades

Cadastro de livros com validação de duplicidade por título
Atualização de dados de um livro por ID
Remoção de livro por título
Consulta de livro por título
Listagem completa do acervo
Respostas HTTP semânticas (201, 200, 204, 400, 404)

Como rodar o projeto 

Pode rodar na própria IDE ou pelo Terminal com o mvn spring-boot:run
